객체는 몇 가지 특수한 기능을 가진 연관 배열(associative array)입니다. 객체는 프로퍼티(키-값 쌍)를 저장합니다. 프로퍼티 키는 문자열이나 심볼이어야 합니다. 보통은 문자열입니다. 값은 어떤 자료형도 가능합니다. 아래와 같은 방법을 사용하면 프로퍼티에 접근할 수 있습니다. 점 표기법: obj.property 대괄호 표기법 obj["property"]. 대괄호 표기법을 사용하면 obj[varWithKey]같이 변수에서 키를 가져올 수 있습니다. 객체엔 다음과 같은 추가 연산자를 사용할 수 있습니다. 프로퍼티를 삭제하고 싶을 때: delete obj.prop 해당 key를 가진 프로퍼티가 객체 내에 있는지 확인하고자 할 때: "key" in obj 프로퍼티를 나열할 때: for (let key in obj) 지금까진 '순수 객체(plain object)'라 불리는 일반 객체에 대해 학습했습니다. 자바스크립트에는 일반 객체 이외에도 다양한 종류의 객체가 있습니다. Array – 정렬된 데이터 컬렉션을 저장할 때 쓰임 Date – 날짜와 시간 정보를 저장할 때 쓰임 Error – 에러 정보를 저장할 때 쓰임
