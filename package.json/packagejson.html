package.json package.json은 프로젝트의 정보를 정의하고, 의존하는 패키지 버전 정보를 명시하는 파일입니다. 프로젝트의 정보 - name, version 영역 패키지 버전 정보 - dependencies 또는 devDependencies 영역 명령어: npm init -y 처음 시작 할때 모듈 :모듈은 node.js의 require() 함수로 로드될 수 있는 node_modules 디렉토리안의 파일 또는 디렉토리입니다.
